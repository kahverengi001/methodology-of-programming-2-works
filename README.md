## Development
#### Getting Started
Clone this repository.

```bash
git clone --recursive https://github.com/kahverengi001/methodology-of-programming-2-works.git
cd methodology-of-programming-2-works
```

##### Compile & Run & Debug

```bash
./gradlew run
```
